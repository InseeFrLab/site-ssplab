Outils et codes

Le SSP Lab met à disposition, dès que c'est possible, le code issu de ses expérimentations, ainsi que des outils qui peuvent etre utilisés largement au sein du SSP.

Pour accéder au code des principales expérimentations, les repos de référence sont ceux utilisés dans l'ensemble de l'Insee. Il s'agit :

  • du GitHub de l'Insee, qui inclut les repos de codes en production
  • du repo InseeFrLab, qui inclut le code d'éléments plus exploratoires. On y retrouve notamment un code du SSP Lab réalisé suite à l'analyse des mouvements de population lors du confinement, à partir des données de téléphonie mobile.

En termes d'outils, le SSP Lab tente de développer des API pour que les résultats des expérimentations. Pour le moment, ces APIs ne sont cependant disponibles que via le réseau interne de l'Insee.
